dbCAN-PUL download index README.txt (by Catie Ausland, causland@niu.edu 06/28/2020)

This current release of dbCAN-PUL is based on PubMed searches as of Jan 8 2020. 

This directory contains 5 subdirecotries and 3 files:

1.characterization/
	Tarball files of PUL data grouped by characterization method. Each tarball file includes datafiles for all PULs verified with the given characterization method.
2.dbCAN-PUL/
	Directory containing data files for all 602 PULs
3.dbCAN-PUL_CGC_vs_PUL_coverage.xlsx
	Excel Workbook detailing which PULs have CGC coverage and of which type (for more information of CGCs and PUL-CGC congruence, please see the 'intro' sheet of the workbook or the Help page on dbCAN-PUL)
4.genera/
	Tarball files of PUL data grouped by genera or metagenome. Each tarball file includes datafiles for all PULs of a given genera/taxon name. 
5.genomes/
	Tarball files of PUL data grouped by species binomial name or metagenome name. Each tarball file includes datafiles for all PULs of a given spcies/taxon name. 
6.PUL.faa
	FASTA format file that contains proteins sequences of all proteins from PULs in dbCAN-PUL and includes annotation information. 
7.README.txt
	The file you are currently reading :) 
8.substrate/
	Tarball files of PUL data grouped by substrate. Each tarball file includes datafiles for all PULs verified to act on a given substrate.
